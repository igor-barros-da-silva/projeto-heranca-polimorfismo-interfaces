Introdução a Interfaces

  • Uma interface pode definir uma série de métodos, mas nunca conter implementação deles.
  • Ela só expõe o que o objeto deve fazer, e não como ele faz, nem o que ele tem.
  • Como ele faz vai ser definido em uma implementação dessa interface.
Lemos a interface da seguinte maneira:

*"quem desejar entrar no sistema precisa obrigatoriamente seguir uma regra dentro so sistema."

Ela é um contrato onde quem assina se responsabiliza a implementar esses métodos.

Problema:
Queremos que apenas o secretário do nosso sistema pode acessar o sistema.
Solução:
O secretário deve implementar uma interface que mostra, obriga e realiza essa liberação.
