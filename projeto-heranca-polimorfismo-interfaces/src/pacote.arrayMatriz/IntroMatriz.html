Introdução a Matrizes

Os vetores bidimensionais ou matrizes são estruturas de dados que representam um conjunto de valores do mesmo tipo.

São referenciados pelos seus indíces assim como os arrays em uma matriz, é igual a uma tabela do Excel onde temos os valores de linhas e colunas, na matriz podemos ter por exemplo uma coluna com indíce 1 e dentro dele 3 valores.

A matriz de dados é usado para agrupamento e organização de dados na estrutura de dados em programação de sistemas.

Entende-se que uma Matriz é um Array onde cada posição contém outro Array.
