Arrays ou Vetore em Java

Nos estudos em progemação de sistemas a parte de estrutura de dados é uma das mais importantes onde aprendemos a manipular os dados em determinadas posições, esse dados podem ser de um tipo, va´rios tipos ou até mesmo uma série de objetos armazenados em determinadas posições.

Ondese aplica o uso de Array?

É muito comum no desenvolvimento de jogos, desenvolvimento de sistema em sua parte gráfica, já viram aqueles equipamentos de laboratório com aqueles gráfico incríveis se comunicando com hardware?
Então você tem um grande uso de Array, outro grande exemplo é que linguagem de programação em determinadas rotina ou processos nós teremos apenas um array(vetor) ou matriz com dados para serem lidos e processados e o conhecimento do programador nessa parte será um enorme diferencial professional.
