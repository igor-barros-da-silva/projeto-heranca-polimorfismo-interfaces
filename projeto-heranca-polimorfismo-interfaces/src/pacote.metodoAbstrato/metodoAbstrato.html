Método Abstrato

  • Existe um recurso em Java que, em uma classe abstrata, podemos escrever que determinado método será sempre escrito pelas classes filhas.
  • Isto é, é um método abstrato.
  • Ele indica que todas as classes filhas(concretas, isto é, que não forem abstratas) devem reescrever esse método ou não compilarão.
  • É como se você herdasse a responsabilidade de ter aquele método.
  • Um método abstrato poder feito somente dentro de uma classe abstrata.
